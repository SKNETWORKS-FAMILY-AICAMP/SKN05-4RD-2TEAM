  <br/>
  
  <h3> 📎 BackGround 📎 </h3>

  <img src="https://github.com/user-attachments/assets/c884a85d-1074-4389-b305-31867252ddb5" width="700" height="500"/>

  <br/>
  
  🏷 **'응급 처치 절차'** 는 '골든 타임' 내에 수행될 경우 생존률을 매우 향상시킬 수 있는 중요한 내용이다.
  
  🏷 이에 따라 '응급 처치'에 관한 교육이 확대되고 있지만, **실제로 이해하고 기억하는 비율은 11.7%에 불과**하다고 한다.
  
  🏷 실제로 갑자기 눈 앞에 응급상황이 펼쳐졌을 때, **곧바로 자신있게 대응할 수 있는 사람은 많지 않을 것**이다.
  
  🏷 이에 따라, 그 자리에 **'응급 처치 절차'에 대해 알려주고**, **원하는 지역의 병원을 알려줄 수 있는 시스템**은 갑자기 발생할 수 있는 상황에서 주변의 소중한 사람을 지켜줄 수 있는 **든든한 조력자**가 될 것이다.

  <h2>📌 Comment 📌</h2>

  ### 🏷 최영민

  파인튜닝 한 LLM 모델을 활용하여 RAG Chain을 구성하고, 이를 Django 웹 어플리케이션에 서빙하며, 이를 AWS 상에 배포해보는 전 과정을 경험할 수 있었던 값진 시간이었던 것 같습니다. 여러 이슈들을 함께 해결해준 팀원들에게 감사합니다! 모두 고생하셨습니다.

  ### 🏷 서장호

  장고 기반 웹 애플리케이션 개발과 배포를 통해 실제 서비스 구현과 운영에 대한 경험을 쌓을 수 있었으며, 문제 해결 능력과 배포 과정에서의 이해도를 높일 수 있었습니다
  
  ### 🏷 김지연

  aws 환경에서 nginx와 gunicorn을 활용해 웹 애플리케이션을 배포하며 서버 구성과 운영을 경험할 수 있는 뜻 깊은 시간이었다.

  ### 🏷 조주영

  제가 이번 프로젝트에서 급한 일로 인해 자리를 비우게 되어 조금이나마 도움이 되지 못해 죄송합니다.  그래도 팀원들이 으쌰으쌰 해줘서 이번 프로젝트는 잘 마무리된것 같습니다. 모두 수고하셨습니댜.
